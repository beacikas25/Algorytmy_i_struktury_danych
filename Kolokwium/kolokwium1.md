Pytanie 'Czy wskazany algorytm jest całkowicie poprawny?': 
Pseudokod: 
```c
sum(array, len) {
    i = 0, sum = 0
    
    while (i < len) {
      sum = sum + array[i];
      i = i + 1;
    }
    
    return sum;
}
```
Odpowiedz: 
        a) Tak
